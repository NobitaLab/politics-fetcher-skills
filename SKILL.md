# 时政抓取 Skill

从粉笔网时政板块抓取本周时政汇总（第一篇文章），更新到笔记。

## 功能说明

1. 访问粉笔网时政列表页
2. 找到第一篇"时政汇总"文章
3. 提取本周时政条目（通常5条）
4. Claude 分析整理后写入笔记

## 执行步骤

### 1. 运行抓取脚本

```bash
cd ~/.claude/skills/时政抓取
node fetch-weekly-politics.js
```

### 2. 确认更新

脚本会自动：
- 启动Edge浏览器
- 访问粉笔网
- 提取时政内容
- 输出 JSON 数据供 Claude 分析

## 输出文件

- 整理内容后更新到笔记 `0.备忘/HomePage/每日时政.md`（vault 相对路径）
- 保留原笔记属性和大体格式
- 更新 date 属性

## 示例输出

```
=== 本周时政 ===
1. 我国加快建设分级诊疗体系
2. 习近平会见西班牙首相
3. 习近平会见阿联酋阿布扎比王储
4. 我国成功发射吉星高分07A02星等8颗卫星
5. 习近平同越共中央总书记、国家主席苏林举行会谈
```

## 技术实现

使用 Puppeteer + Edge浏览器抓取动态网页内容。
